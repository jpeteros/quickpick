# quickpick

A React inventory dashboard for a vending machine that lets you manage products, update stock, edit items, and delete entries.

## Run locally

1. Install dependencies: `npm install`
2. Start the backend API: `node server.js`
3. Start the frontend: `npm run dev`

The app will be available at http://localhost:5174/ and the API at http://localhost:3001/api/items.
